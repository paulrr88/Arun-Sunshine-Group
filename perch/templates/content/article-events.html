Instructions:

This is for the events page, with each section here representing one event.

You can either:
- Edit an existing event by changing the fields that are currently on this page
- Add a new event to the bottom of the list by scrolling to the bottom of this page and clicking "Save & Add another", which will result in a new blank set of fields appearing at the bottom of this page that can then be filled in with appropriate text.

You can also delete an event by clicking the small red delete button to the top right of an event entry.

The main part of the event article should be placed in the large text box, and you can find various buttons that will allow you formatting options. The editor uses "Markdown", which is basically just using symbols and text to do what a text editor like Word would typically do. You can just highlight text and press the buttons and the Markdown will be added automatically.

If you want to add an image:

- Position the writing cursor in the text where you want the image to appear
- Click the little landscape symbol in the line of tools above the text editing area (it will say Picture if you hold your mouse cursor over it for a bit)
- In the box that appears, click Browse...
- This will bring up a box that shows your hard drive, now you can navigate to the folder with the desired image, and click Open to upload it.

This will create a line of Markdown that looks something like this: [cms:asset 60]. You can cut and paste this if you want to move it around your writing and reposition it.

If you want to have multiple images side by side, simply move the above code into the same line with NO SPACE inbetween, so:
[cms:asset 60][cms:asset 61]. This should happen automatically if you just put one image in after another without moving the cursor. I recommend having only two images to a line, so:

[cms:asset 60][cms:asset 61]
[cms:asset 62][cms:asset 63]

Otherwise the layout will start to go wrong!

If you want to have larger text, use the H symbol and select between headings 1-6 (1 is largest, 6 is smallest).

As with everything else in Perch, remember to click Save Changes at the bottom to save your text to the page!
