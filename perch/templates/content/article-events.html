Instructions:

This is for the events page, with each section here representing one event.

You can either:
- Edit an existing event by changing the fields that are currently on this page
- Add a new diary entry to the bottom of the list by scrolling to the bottom of this page and clicking "Save & Add another", which will result in a new blank set of fields appearing at the bottom of this page that can then be filled in with appropriate text.

You can also delete an event by clicking the small red delete button to the top right of an event entry.

The main part of the event article should be placed in the large text box, and you can find various buttons that will allow you formatting options. The editor uses "Markdown", which is basically just using symbols and text to do what a text editor like Word would typically do. You can just highlight text and press the buttons and the Markdown will be added automatically.

If you want to add an image:

This will create a line of Markdown that looks something like this: ![image](/directory/image.jpg)

If you want to have multiple images side by side
